# Aula 4 - Ambientes Reprodutíveis com Docker

## Objetivos da Aula
- Compreender os conceitos de containerização
- Aprender a criar e gerenciar containers Docker
- Desenvolver ambientes reprodutíveis para projetos de dados
- Entender Docker Compose para orquestração de serviços

1. Introdução à containerização
2. Conceitos básicos do Docker
3. Criação de Dockerfiles
4. Gerenciamento de imagens e containers
5. Docker Compose
6. Boas práticas para ambientes de Data Science
