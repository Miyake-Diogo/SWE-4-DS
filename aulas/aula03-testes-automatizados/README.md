# Aula 3 - Testes Automatizados

## Objetivos da Aula
- Compreender a importância de testes automatizados
- Conhecer diferentes tipos de testes (unitários, integração, E2E)
- Aprender a escrever testes eficazes em Python
- Entender Test-Driven Development (TDD)

## Conteúdo Programático
1. Introdução a testes automatizados
2. Tipos de testes: unitários, integração e end-to-end
3. Frameworks de teste em Python (pytest, unittest)
4. Test-Driven Development (TDD)
5. Cobertura de código
6. Testes para projetos de Machine Learning
