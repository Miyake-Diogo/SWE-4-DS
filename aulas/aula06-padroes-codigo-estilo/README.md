# Aula 6 - Padrões de Código e Estilo

## Objetivos da Aula
- Compreender a importância de padrões de código
- Conhecer guias de estilo (PEP 8, Google Style Guide)
- Aprender a utilizar linters e formatadores
- Implementar revisão de código eficaz

## Conteúdo Programático
1. Importância de padrões de código
2. PEP 8 e guias de estilo Python
3. Linters (pylint, flake8, ruff)
4. Formatadores automáticos (black, autopep8)
5. Type hints e validação estática (mypy)
6. Code review e boas práticas

## Materiais de Apoio
- Slides
- Exemplos de código
- Exercícios práticos
- Leituras recomendadas
