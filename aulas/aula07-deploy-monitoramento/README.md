# Aula 7 - Deploy e Monitoramento de Código

## Objetivos da Aula
- Compreender estratégias de deploy de aplicações
- Conhecer práticas de CI/CD (Continuous Integration/Continuous Deployment)
- Aprender a monitorar aplicações em produção
- Implementar pipelines automatizados

## Conteúdo Programático
1. Conceitos de deploy e entrega contínua
2. CI/CD com GitHub Actions, GitLab CI
3. Estratégias de deploy (blue-green, canary, rolling)
4. Monitoramento e observabilidade
5. Logs, métricas e alertas
6. Deploy de modelos de Machine Learning
