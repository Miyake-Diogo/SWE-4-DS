# Aula 2 - Git e Controle de Versão

## Objetivos da Aula
- Compreender os conceitos de controle de versão
- Dominar os comandos básicos e avançados do Git
- Aprender boas práticas de versionamento de código
- Entender workflows de colaboração com Git

## Conteúdo Programático
1. Introdução ao controle de versão
2. Comandos básicos do Git (init, add, commit, push, pull)
3. Branches e merge
4. Resolução de conflitos
5. Git workflows (Git Flow, GitHub Flow)
6. Boas práticas para commits e mensagens

- Exercícios práticos
