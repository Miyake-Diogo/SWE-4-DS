# Aula 5 - Gerenciamento de Dependências

## Objetivos da Aula
- Compreender a importância do gerenciamento de dependências
- Conhecer ferramentas de gerenciamento (pip, poetry, conda)
- Aprender a criar ambientes virtuais isolados
- Entender versionamento de dependências

## Conteúdo Programático
1. Importância do gerenciamento de dependências
2. Ambientes virtuais (venv, virtualenv, conda)
3. Pip e requirements.txt
4. Poetry para gerenciamento avançado
5. Conda para ambientes de Data Science
6. Boas práticas para versionamento de dependências
