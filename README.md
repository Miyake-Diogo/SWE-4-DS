# SWE-4-DS
Repositorio para a Disciplina de Engenharia de Software para Cientistas de Dados para o curso de POSTECH da FIAP PosTech Machine Learning Engineering (MLET)

## Estrutura do Curso

Este repositório contém materiais didáticos organizados por aulas:

### [Aula 1 - Fundamentos de Engenharia de Software](aulas/aula01-fundamentos-engenharia-software/README.md)
Conceitos fundamentais de Engenharia de Software aplicados à Ciência de Dados.

### [Aula 2 - Git e Controle de Versão](aulas/aula02-git-controle-versao/README.md)
Controle de versão com Git, comandos essenciais e workflows de colaboração.

### [Aula 3 - Testes Automatizados](aulas/aula03-testes-automatizados/README.md)
Testes unitários, de integração e TDD para projetos de Data Science.

### [Aula 4 - Ambientes Reprodutíveis com Docker](aulas/aula04-ambientes-reprodutiveis-docker/README.md)
Containerização e criação de ambientes reprodutíveis usando Docker.

### [Aula 5 - Gerenciamento de Dependências](aulas/aula05-gerenciamento-dependencias/README.md)
Gerenciamento de dependências e ambientes virtuais em Python.

### [Aula 6 - Padrões de Código e Estilo](aulas/aula06-padroes-codigo-estilo/README.md)
Padrões de código, linters, formatadores e boas práticas.

### [Aula 7 - Deploy e Monitoramento de Código](aulas/aula07-deploy-monitoramento/README.md)
CI/CD, deploy de aplicações e monitoramento em produção.

### [Aula 8 - Refatoração de Projeto DS](aulas/aula08-refatoracao-projeto-ds/README.md)
Técnicas de refatoração e arquitetura para projetos de Data Science.
